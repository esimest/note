下载mysql仓库：sudo yum localinstall https://repo.mysql.com//mysql80-community-release-el7-1.noarch.rpm -y
安装：sudo yum install mysql-community-server -y

查找root临时密码：sudo grep 'temporary password' /var/log/mysqld.log

登录后修改密码：ALTER USER 'root'@'localhost' IDENTIFIED BY '密码';大小写数字特殊符号
修改密码强度和长度限制：set global validate_password.policy = 0 ;set global validate_password.length=6;
set global validate_password.policy = medium ;set global validate_password.length=8;

修改/etc/my.cnf
添加
[client]
host = 
user = 
password = 
port =
可在登录时直接输入mysql无需指定用户和输入密码

添加用户 赋权限

use mysql;

CREATE user 'username' IDENTIFIED BY 'password';

GRANT ALL privileges ON database.tables TO 'username'@'ip(default=%)';
GRANT EXECUTE, INSERT, SELECT, UPDATE ON world.* TO 'chengjian'@'%';


删除用户 DROP user username
