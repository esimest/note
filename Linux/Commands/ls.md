# ls 笔记

> ls 后不接文件或目录，默认参数为 .
> ls -d 显示的结果为 .
> `ls * == ls ./*`

## format

> ls 默认以行输出结果，即如果目录下有 6 个文件，$(ls |wc -l) == 6
> 可以使用 -C 选项指定以列格式输出结果

```shell
# 每行显示一个文件/目录
ls -1

# 显示文件/目录的大小
ls -s(size)

# 显示文件/目录大小，单位为 Kb
ls -sk

# 以后缀的形式显示结果(路径加'/' 后缀，可执行文件以'*'为后缀..)
ls -F

# 直显示目录名，不列出目录内容
ls -d
```
